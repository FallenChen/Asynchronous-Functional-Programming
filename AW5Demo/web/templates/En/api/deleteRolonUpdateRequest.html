A delete Rolon update request is an update request which deletes an existing Rolon. This request is created by calling deleteRolonUpdateRequest factory method on the extended context object. Once created, the request can then be added to the update requests list for subequent processing.

Factory Arguments

The factory method in the extended context requires three arguments:

String requestName
The name of the update request. Each update request must have a unique request name. The request name is used as a prefix for any results added to the update parameters map when the request is handled.
Context context
the context map.
RolonProxy rolonProxy
a Rolon proxy representing the Rolon to be deleted.
String viewName
The name of the view performing the update.
