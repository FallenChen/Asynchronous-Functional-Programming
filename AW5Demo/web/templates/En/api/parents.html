A parents object is loaded to the context map with a name of {timestamp}|{rolonUuid}.parents when a Rolon is read. This object holds a list of the Rolon's parents, together with the names by which these parents reference the child Rolon.

map<String,Integer>

Maps the parent UUIDs to their position.

String

role(int n)

Returns the role name of the nth parent.

String

uuid(int n)

Returns the uuid of the nth parent.

String

childName(int n)

Returns the name used by the nth parent for referencing this child Rolon.
