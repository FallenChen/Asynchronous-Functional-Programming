Update Parameters is a Map[String,Any] used in processing updates and is accessible via the extended context.

Special Update Parameters

There are 4 special update parameters which have side-effects:

updateRequestInError
This parameter is set while processing the update requests to the name of the failing update request. when an error occurred.
templateRequest
Replaces the pathname of the request from the browser.
user.language
Changes the language selected by the user.
newUser.rolonUuid
Effects a user login. Set to "anonymous" to logoff.

The templateRequst, user.language and user.uuid parameters have no effect however when there are no update requests, or when there was an error while processing the update requests.
