The Context class provides access to the context map used by the template engine. An instance of this class is bound to every Groovy script with the name context.

boolean

contains(String name)

Returns true when the context map has a value with the given name.

String

get(String name)

Returns the value assigned to the given name as a String. When there is no value assigned to the given name, an empty String is returned.

StringBuilder

getVar(String name)

Returns the StringBuilder object of the named variable.

setCon(String name, String value)

Adds the constant value to the context map with the given name.

newVar(String name, String value)

Adds a StringBuilder object to the context map with the given name and initialized to the given value.

boolean

isVar(String name)

Returns true when the name is present in the context map and the value is a StringBuilder object.

setVar(String name, String value)

Replaces the contents of the named StringBuilder object with the given value.

makeVar(String name, String value)

If the value of the given name is a StringBuilder object, then the contents of the StringBuilder is replaced with the given value; otherwise a new StringBuilder object is created with the given name and value.

java.util.TreeMap<String,String>

stringMap(String prefix)

Returns a TreeMap of names and values for all the names in the context map which begin with the given prefix, only the prefix is removed from the names as they are added to the TreeMap.

The prefix may be an empty String, in which case the contents of the entire context map are added to the TreeMap. When the prefix is not an empty String, a "." is added to the end of the prefix prior to processing the context map.

java.util.TreeSet<String>

stringSet()

Returns a TreeSet of all the names in the context map.

setSpecial(String name, Object value)

Adds an object to the context map with the given name.

Object

getSpecial(String name)

Returns the object in the context map with the given name.
