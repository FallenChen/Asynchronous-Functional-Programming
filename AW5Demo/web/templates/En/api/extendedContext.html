java.util.List<UpdateRequest> updateRequests

The update requests to be processed.

java.util.Set<String> reservedAttributes

The reserved attributes can not be used in an application:

  • btreeSize,
  • elementType,
  • lastcommand,
  • name,
  • password and
  • rolonType.

java.util.Map<String,Object> updateParameters

Parameters to be used in processing the update requests, these parameters are added to a new context map on completion of the update request processing.

assignViewParameters(Context context)

Used by update requests to add the required view parameters.

Role

role(String roleName)

Returns the named role, or null if there is no such role.

String

singletonUuid(String roleName)

If the named role is a singleton, then the UUID of the Rolon is returned; otherwise the returned value is not defined. I.E. For non-singleton roles, the returned value may be null or may be the UUID of a non-existing rolon.

RolonProxy

rolonProxy(Context context, String uuid)

Returns a new RolonProxy created from a Rolon already present in the context map.

RolonProxy

userProxy(Context context)

Returns a new RolonProxy created from the user Rolon.

RolonProxy

singletonProxy(Context context, String roleName)

Returns a new RolonProxy created from a Rolon already present in the context map.

java.util.HashSet<String>

userGroups(Context context, String String privilege)

Returns a set of the UUIDs of the groups for which the user has a membership with the specified privilege. And when the user is a member of a nested group, the UUIDs of the super groups of that nested group are included.

boolean

hasPrivilege(Context context, String rolonUuid, String privilege)

Returns true when the user has the specified privilege for the given rolon.

The rolon be already loaded into the context map and the privilege must be reader, writer or owner.

NoopUpdateRequest

noopUpdateRequest(String requestName, Context context)

Returns a new NoopUpdateRequest.

CreateChildUpdateRequest

createChildUpdateRequest(String requestName, Context context, RolonProxy parentProxy, String childRoleName, String childName, String viewName)

Returns a new CreateChildUpdateRequest. But if the parent role does not permit a child of the given role or if the child role does not permit a parent of the given role, then a IllegalArgumentException is thrown.

RolonUpdateRequest

rolonUpdateRequest(String requestName, Context context, RolonProxy rolonProxy, String viewName)

Returns a new RolonUpdateRequest.

DeleteRolonUpdateRequest

deleteRolonUpdateRequest(String requestName, Context context, RolonProxy rolonProxy, String viewName)

Returns a new DeleteRolonUpdateRequest.

RenameRolonUpdateRequest

renameRolonUpdateRequest(String requestName, Context context, RolonProxy parentProxy, RolonProxy childProxy, String newChildName, String viewName)

Returns a new RenameRolonUpdateRequest.

java.util.NavigableSet<String>

propertyNames()

Returns the set of property names.

String

invertTimestamp(String timestamp)

Inverts the timestamp.

SequenceConvenience

groovySequence(Context context, String groovySequenceCommandName)

Returns the SequenceConvenience created by a Groovy sequence command. The command name would be in the form "children" rather than in the form "seq_children.groovy".

SequenceConvenience

navigableSequence(NavigableSet<String> set, boolean reverse)

Returns a sequence over a navigable set of strings. When reverse is true, the order is reversed.

SequenceConvenience

intersectionSequence(List<SequenceConvenience> sequences, boolean reverse)

Returns a sequence which is the intersection of the list of sequences. The reverse flag must agree with the value of the reverse flag of the sequences in the provided list.

SequenceConvenience

unionSequence(List<SequenceConvenience> sequences, boolean reverse)

Returns a sequence which is the union of the list of sequences.

SequenceConvenience

subSequence(SequenceConvenience wrappedSequence, String prefix)

Returns a sequence containing only those items from the wrapped sequence which start with the provided prefix.

SequenceConvenience

prefixesSequence(SequenceConvenience wrappedSequence, char delimiter)

Returns a sequence of the unique prefixes found in the wrapped sequence.

SequenceConvenience

invertedTimestampSequence(SequenceConvenience wrappedSequence)

Inverts the keys of the wrapped sequence.

SequenceConvenience

contextFilterSequence(Context context, SequenceConvenience wrappedSequence, String filterCommandName)

Returns a sequence of selected items.

SequenceConvenience

contextFilterSequence(Context context, SequenceConvenience wrappedSequence, String filterCommandName, char delimiter)

Returns a sequence of selected items.

SequenceConvenience

qualifiedSequence(String timestamp, String uuid)

Returns a sequence of the qualified rolon UUIDs for the given Roloon and time.

SequenceConvenience

childrenSequence(String timestamp, String uuid)

Returns a sequence consisting of the child name, the character 5 and the UUID of the child rolon.

SequenceConvenience

parentsSequence(Parents parents)

Returns a sequence consisting of the parent UUID, the character 5 and the child name used by the parent to reference this child.

SequenceConvenience

templateFilterSequence(Context context, SequenceConvenience wrappedSequence, String templatePath)

Uses a template to filter the wrapped sequence.

SequenceConvenience

templateFilterSequence(Context context, SequenceConvenience wrappedSequence, String templatePath, char delimiter)

Uses a template to filter the wrapped sequence. Key extension is also supported.

SequenceConvenience

journalEntryTimestampSequence(Context context)

Returns a sequence of all journal entry Rolons.

SequenceConvenience

journalEntryTimestampSequence(Context context, String uuid)

Returns a sequence of all journal entry Rolons effecting a given Rolon.

SequenceConvenience

keySequence(TreeMap<String, Object> tm)

Returns a sequence of the keys in the given tree map.

Object

getValue(TreeMap<String, Object> tm, String key)

Returns the value for a given key in the given tree map.
