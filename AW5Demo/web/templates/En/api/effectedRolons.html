An effected rolons object is a TreeMap<String, TreeMap<String, TreeMap<String, String>>> and is loaded to the context map with a name of {timestamp}|{rolonUuid}.effectedRolons when a Rolon is read.

When the rolon is a Journal Entry, the primary keys of this map are the effected Rolon UUIDs, the secondary keys are change sequence numbers and the lowest level maps are name/value pairs which describe the effected changes.
