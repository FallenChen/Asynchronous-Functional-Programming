The sitemap query looks into all dir.prop files and add it's contents in context map.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="sitemap"

Result

This command places contents of dir.prop in context map with prefix "sitemap".If no dir.prop found, then no updates are made to the context map.

Example

Here's the code to access the ark rolon: 

    <aw:query cmd="sitemap">
      <aw:include path="dump.xml" />
    </aw:query>
