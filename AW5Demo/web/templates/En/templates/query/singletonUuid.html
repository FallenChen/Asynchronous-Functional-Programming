The singletonUuid query look for a singleton rolon associated with a role. It return the uuid of the rolon associated with role.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="singletonUuid"

Parameters

This command uses one parameter which must be present in the context map:

The role parameter
provides the name of the existing role.

Result

This command places one result in the context map:

The rolonUuid
is added to the context map with a value of rolon uuid.

Example

Here's the code to access the ark rolon: 

    <aw:query cmd="singletonUuid" role="home">
      <p>
        <b>Ark Rolon:</b>
        <aw:contents name="rolonUuid" />
      </p>
    </aw:query>
