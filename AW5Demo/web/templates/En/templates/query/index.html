Query Commands

The aw:query element provides a means for invoking query commands. AW5 includes a number of query commands, and additional commands can be provided by application developers.

The query commands invoked by this element may query the AW5 database and other data sources, as well as using the context map to access parameters and return results.

Query Attributes

The aw:query element has a single required attribute, cmd, which must be set to the name of the query command to be invoked.

As a convenience, all aw:query attributes are added as string contents to the context map, which makes them accessible to the query command being invoked.
