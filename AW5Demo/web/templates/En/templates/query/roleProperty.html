The role property query command returns the value of a role property.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="roleProperty"

Parameters

This command uses two parameters:

The role parameter
provides the name of the role. This is a required parameter and must be present in the context map.
The pname parameter
provides the name of a role property. This is a required parameter and must be present in the context map.
The name parameter
provides the name of the return value. This is a required variable and must be present in the context map. The given name is assigned the value of the selected role property. If the selected property is not present in the role properties map, then no change is made to the context map.
