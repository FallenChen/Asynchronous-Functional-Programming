The property query command returns the value of a configuration property.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="property"

Parameters

This command uses two parameters:

The pname parameter
provides the name of a configuration property. This is a required parameter and must be present in the context map.
The name parameter
provides the name of the return value. This is an optional parameter and defaults to "property". The given variable is assigned the value of the selected configuration property. If the selected property is not present in the configuration properties map, then no change is made to the context map.
