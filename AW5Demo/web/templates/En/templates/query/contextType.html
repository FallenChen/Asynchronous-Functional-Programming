The contextType query command determines the type of a value in the context map.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="contextType"

Parameters

This command takes a single parameter, name, which identifies a value in the context map.

Results

This command adds a constant named type to the context map. The constant is either variable, constant or special.
