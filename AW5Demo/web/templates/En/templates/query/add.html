The add query command performs arithmetic addition of two integers.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="add"

Parameters

This command uses two parameters which must be present in the context map:

The name parameter
provides the name of an existing integer variable in the context map. This is the variable which the add command updates with the result of addition.
The value parameter
provides the integer, which is added to the named variable. If not present, 1 is taken as default.
