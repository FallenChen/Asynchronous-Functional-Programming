The append query command appends a string to the current value of a variable.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="append"

Parameters

This command uses two parameters which must be present in the context map:

The name parameter
provides the name of an existing string variable in the context map. This is the variable which the append command updates.
The value parameter
provides a string, which is appended to the named variable.
