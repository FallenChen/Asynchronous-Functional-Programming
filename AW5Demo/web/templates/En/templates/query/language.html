The language query command converts a language code into the language name. It does this by returning the value of the langName property from the dir.prop file in the directory named by the language code. (The names of the subordinate directories under the templates directory are the language codes)

This command also creates a flag in the context map when the given language is the user's current language.

Attributes

The cmd attribute on the aw:query element is required to invoke this command: 

cmd="language"

Parameters

The langCode parameter provides the language code and must be present in the context map.

Results

This command places two results in the context map:

The langName property
from the selected dir.prop file is added to the context map.
When the language matches the user's current language
selected is added to the context map with a value of "selected".
