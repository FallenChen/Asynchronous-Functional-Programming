The aw:rolon element is transformed into the code needed to navigate to another Rolon. This element should only be used within a view.

In the generated code, the contents of the aw:rolon element is nested inside an "a" element, with the effect that it becomes clickable by the user.

A single parameter is used, uuid, which is set to the UUID of the Rolon being navigated to.
