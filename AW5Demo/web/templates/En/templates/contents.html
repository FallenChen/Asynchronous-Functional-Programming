The aw:contents element is used to display a string, generally a string value from the context map. It takes either a value attribute or a name attribute. 

<p setcon.key="foo.bar" foo.bar="xyzzy">
	1. <aw:contents value="key" />
   	2. <aw:contents aw:value="key" />
    3. <aw:contents aw-rmpre:value="key" />
    4. <aw:contents name="key" />
    5. <aw:contents aw:name="key" />
</p>

The above generates the following HTML: 

<p>
	1. key
    2. foo.bar
    3. bar
    4. foo.bar
    5. xyzzy
</p>
