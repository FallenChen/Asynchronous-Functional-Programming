The attributes sequence command supports looping through the attributes of a Rolon and is invoked using the aw:loop element. The series of keys generated by this command are the names of the child Rolons.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="attributes"

Parameters

The children sequence command requires one parameter, whose name may vary depending on the value of the loopPrefix parameter used by the aw:loop element:

rolonUuid
The UUID of the Rolon.
