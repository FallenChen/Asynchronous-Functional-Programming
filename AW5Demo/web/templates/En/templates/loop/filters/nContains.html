The nContains filter command is used to prevent further processing of keys which contains text. To know how the groovy script works step by step, please see this example

Attributes

Filters use 2 attributes specified on the aw:loop element:

The filter attribute on the aw:loop element
is required to invoke this command:

filter="nContains"

The loopPrefix
when specified as an attribute on the aw:loop element, is pre-pended to the parameters used by the nContains filter command.

Properties

The text string is used by this command to determine which keys are not to be processed further. Note that the name of this property is modified when a loopPrefix is specified.

Result

This filter excludes all the properties which contains text.
