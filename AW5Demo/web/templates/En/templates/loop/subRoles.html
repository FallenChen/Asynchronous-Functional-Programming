The subRoles sequence command supports looping through the sub-roles of a given role and is invoked using the aw:loop element. The series of values generated by this command are the names of the sub-roles.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="subRoles"

Parameters

The subRoles sequence command requires one parameter, role, which must be the name of a valid role.

Example

Here's the code: 

    <aw:loop cmd="subRoles" role="qualifier">
      <p><aw:contents name="key" /></p>
    </aw:loop>
