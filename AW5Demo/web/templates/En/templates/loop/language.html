The language sequence command supports looping through the subdirectories under the templates directory and is invoked using the aw:loop element. The series of values generated by this command are the names of the subdirectories which have a dir.prop file.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="language"

Example

Here's the language.xml template, which displays a set of languages to choose from: 

<p>
  <aw:is  name="_.currentRequest">
    <aw:update cmd="language" />
  </aw:is>
  <aw:nis  name="_.currentRequest">
    <form aw:action="activeTemplate" method="post">
      <select name="language">
        <aw:loop cmd="lang">
          <aw:query cmd="lang" aw:setcon.langCode="key">
            <aw:is name="selected">
              <option aw:value="key" selected="selected">
                current: <aw:contents name="langName"/>
              </option>
            </aw:is>
            <aw:nis name="selected">
              <option aw:value="key">
                <aw:contents name="langName"/>
              </option>
            </aw:nis>
          </aw:query>
        </aw:loop>
      </select>
      <input type="submit" value="Select"/>
      <input type="hidden" name="currentRequest" aw:value="currentRequest"/>
    </form>
  </aw:nis>
</p>

This template is invoked on the header.xml as follows: 

<aw:include path="language.xml"/>
