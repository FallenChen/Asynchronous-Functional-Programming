The superRoles sequence command supports looping through the super-roles of a given role and is invoked using the aw:loop element. The series of values generated by this command are the names of the super-roles.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="superRoles"

Parameters

The superRoles sequence command requires one parameter, role, which must be the name of a valid role.

Example

Here's the code: 

    <aw:loop cmd="superRoles" role="user">
      <p><aw:contents name="key" /></p>
    </aw:loop>
