The properties sequence command supports looping through the configuration properties and is invoked using the aw:loop element. The series of values generated by this command are the names of the configuration properties.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="properties"

Example

Here's the pdump.xml template which displays the configuration properties: 

<p newvar.pdump.resetLink="" newvar.pdump.previousLink="" newvar.pdump.nextLink="">
  <table border="1">
    <tr>
      <th>
      </th>
      <th>
        Property
      </th>
      <th>
        Value
      </th>
    </tr>
    <aw:loop cmd="properties" loopPrefix="pdump">
      <tr aw:class="pdump.stripeClass">
        <td align="right">
          <aw:contents name="pdump.index" />
        </td><td>
          <aw:contents name="pdump.key" />
        </td><td>
          <aw:query cmd="property" aw:pname="pdump.key">
            <aw:contents name="property" />
          </aw:query>
        </td>
      </tr>
    </aw:loop>
  </table>
  <aw:is name="pdump.previousLink">
    <a aw:href="pdump.resetLink">Reset</a>
    <a aw:href="pdump.previousLink">Previous</a>
  </aw:is>
  <aw:is name="pdump.nextLink">
    <a aw:href="pdump.nextLink">Next</a>
  </aw:is>
</p>
