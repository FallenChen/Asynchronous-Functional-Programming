The role properties sequence command supports looping through the properties of a role and is invoked using the aw:loop element. The series of values generated by this command are the names of the role properties.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="roleProperties"

Parameters

This command requires a single parameter, role, which names the role. The name of this parameter will vary depending on the aw:loop parameter, loopPrefix.
