The qualification attribute sequence command supports looping through the qualifiers of a Rolon for which there is a given attribute. This command is invoked using the aw:loop element. The series of keys generated by this command are the UUID's of the selected qualifier Rolons. The value of the given attribute is also provided.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="qualificationAttribute"

Parameters

The qualification attribute sequence command requires two parameters, rolonUuid and qualificationAttribute, where the rolonUuid parameter provides the uuid of a Rolon that has been loaded into the context map and the qualificationAttribute specifies the attribute of interest. The names of both parameters are prepended with the value of loopPrefix.

Loop Constants

The value constant is set to the value of the qualification attribute. (The name of this constant will vary, depending on the value of the loopPrefix parameter.)

Example

In this example we list the qualifiers of the home rolon with a access attribute. 

      <aw:query cmd="singletonUuid" role="users">
        <aw:query cmd="rolon">
          <table border="1">
            <tr>
              <th></th>
              <th>
                Qualifier UUID
              </th>
              <th>
                Qualifier Role
             </th>
              <th>
                Qualification Attribute Value
             </th>
            </tr>
            <aw:loop cmd="qualificationAttribute" setcon.qualificationAttribute="access">
              <tr aw:class="stripeClass">
                <td align="right">
                  <aw:contents name="index" />
                </td><td>
                  <aw:contents name="key" />
                </td><td>
                  <aw:query cmd="versionId">
                    <aw:contents aw.versionId:value="role" />
                  </aw:query>
                </td><td>
                  <aw:contents name="value" />
                </td>
              </tr>
            </aw:loop>
          </table>
        </aw:query>
      </aw:query>
