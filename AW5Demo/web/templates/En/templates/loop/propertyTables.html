The property tables sequence command supports looping through the configuration properties and is invoked using the aw:loop element. The series of values generated by this command are the names of the various tables in the configuration properties.

Attributes

The cmd attribute on the aw:loop element is required to invoke this command: 

cmd="propertyTables"

Example

Here's the template code for displaying the property tables: 

    <table>
      <aw:loop cmd="propertyTables" filter="nContains" text="role">
        <tr><td>
          <aw:contents name="key" />
        </td></tr>
      </aw:loop>
    </table>
