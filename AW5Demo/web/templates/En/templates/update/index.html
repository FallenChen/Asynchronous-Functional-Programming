Query Commands

The aw:update element provides a means for invoking update commands. AW5 includes a number of update commands, and additional commands can be provided by application developers.

The update commands invoked by this element may update the AW5 database and other data sources, as well as using the context map to access parameters and return results.

Update Attributes

The aw:update element has a single required attribute, cmd, which must be set to the name of the update command to be invoked.

As a convenience, all aw:update attributes are added as string contents to the context map, which makes them accessible to the update command being invoked.
