The language update command changes the language selection for a user.

Attributes

The cmd attribute on the aw:update element is required to invoke this command: 

cmd="language"

Request Parameters

The language update command takes two request parameters:

The _.currentRequest parameter
preserves the URL of the last page displayed. On completion of the command an equivalent page, if available, is displayed.
The _.langName parameter
provides the user's new language selection.
