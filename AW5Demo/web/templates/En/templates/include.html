The aw:include element is used to add the content of a template file to the content of a page or to the content of another template. The path attriube is used to identify the template to be included, and can be either a relative or absolute pathname. Here is an example using an absolute pathname: 

<aw:include path="/templates/head.xml" setcon.title="aw:include element" />

Here is the contents of the head.xml file: 

<head> 
	<title><aw:contents name="title" /></title>
    <META http-equiv="Pragma" content="no-cache" /> 
    <META http-equiv="Cache-Control" content="no-cache,no-store" /> 
</head>

And here is the code generated from the aw:include element: 

<head> 
	<title>aw:include element</title>
    <META http-equiv="Pragma" content="no-cache" /> 
    <META http-equiv="Cache-Control" content="no-cache,no-store" /> 
</head>

Here then is an example of aw:include using a relative pathname: 

<aw:include path="../header.xml" setcon.heading="The aw:include Element" />

Here is the contents of the header.xml file: 

<h1>
	<a href="/"><img src="/images/AgileWiki-logo.png" /></a> 
    <aw:contents name="heading" />
</h1>

And here is the generated HTML: 

<h1>
	<a href="/"><img src="/images/AgileWiki-logo.png" /></a> 
    The aw:include Element
</h1>
