The aw:a element is transformed into a a element, with all but the href attribute left unchanged. The href attribute is also left unchanged unless it is a relative URL, in which case it is converted to an absolute URL.

<aw:a href="index.html">back</aw:a>

The above example is transformed into the following HTML:

<a href="/templates/En/templates/index.html">back</a>

E.G. back

The aw:a element is helpful, as it allows web developers to use URLs which are relative to the template which contains the element - which might not be in the same directory where the generated HTML page is located.
