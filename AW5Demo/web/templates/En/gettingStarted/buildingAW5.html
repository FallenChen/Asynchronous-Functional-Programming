This page describes how to build the AgiliWiki5 source code.

In order to build Agiliwiki firstly open a terminal or command prompt and navigate to the Agilewiki5 directory. Next issue the following command:

mvn clean install - This will perform a fresh compile of the source code, run any unit tests and install the resulting artifacts into your local Maven repository.

As part of this task Maven will automatically download any depdendenices required by Agilewiki, once the entire process is complete you will be presented with a 'BUILD SUCCESSFUL' message.
